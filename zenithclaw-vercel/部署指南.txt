🚀 两种部署方式：

═══════════════════════════════════════════════
方式一：网页部署（最简单，推荐）
═══════════════════════════════════════════════

1. 访问 https://vercel.com
2. 注册/登录账号（免费）
3. 点击 "Add New" → "Project"
4. 拖拽整个 "zenithclaw-vercel" 文件夹到上传区域
5. 等待上传完成（约1分钟）
6. 点击 "Deploy" 按钮
7. 等待2-3分钟，部署完成！

你会得到一个链接：https://zenithclaw-xxx.vercel.app

═══════════════════════════════════════════════
方式二：命令行部署（需先安装工具）
═══════════════════════════════════════════════

1. 打开新的 PowerShell 窗口（管理员模式）

2. 安装 Vercel CLI（只需一次）：
   npm install -g vercel

3. 进入项目目录：
   cd D:\zenithclaw-vercel

4. 登录 Vercel：
   vercel login

5. 部署到生产环境：
   vercel --prod
