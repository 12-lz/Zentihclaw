1

登录 Vercel

  1. 打开浏览器，访问：https://vercel.com
  2. 点击右上角的 "Login" 或 "Sign Up"
  3. 选择登录方式（推荐使用 GitHub 账号登录）
  4. 如果还没有账号，先注册一个（完全免费）
2

创建新项目

  1. 登录后，点击右上角的 "Add New..."
  2. 选择 "Project"
  3. 你会看到项目导入页面
3

上传文件夹

⚠️ 重要：上传整个文件夹，不要单独上传文件！
  1. 找到上传区域（显示 "Import Git Repository" 或类似文字）
  2. 向下滚动，找到 "Upload a file or folder" 链接
  3. 点击后，会打开文件选择窗口
D:\zenithclaw-vercel\
  1. 导航到上述路径
  2. 选中整个 zenit
  3. 点击 "上传" 或 "选择文件夹"

⚙️ vercel.json 0.15 KB
4

配置项目

  1. 上传完成后，你会看到项目配置页面
  2. Project Name: 输入项目名称（例如：zenithclaw）
  3. Framework Preset: 选择 "Other" 或 "Static"
  4. Root Directory: 留空或选择 "./"
  5. Build Command: 留空（静态网站无需构建）
  6. Output Directory: 留空或输入 "./"
💡 提示：大多数情况下，Vercel 会自动检测配置，你只需要点击 "Deploy"
5

部署项目

  1. 检查所有设置是否正确
  2. 点击底部的 "Deploy" 按钮
  3. 等待部署完成（通常 2-3 分钟）
  4. 你会看到部署进度条和日志
6

访问你的网站

  1. 部署完成后，你会看到成功消息
  2. 点击访问链接（类似：https://zenithclaw.vercel.app）
  3. 你的网站现在全球可访问了！🎉

常见问题

Q: 上传后找不到图片文件？

A: 确保上传的是整个文件夹，不是单独的文件。在 Vercel Dashboard 的 Files 标签中应该能看到 4 个文件。

Q: 部署后图片还是不显示？

A: 运行 "D:\zenithclaw-vercel\运行修复工具.bat" 然后重新部署。

Q: 显示 "Build failed" 错误？

A: 确保 Root Directory 和 Output Directory 设置正确，或者留空让 Vercel 自动检测。
